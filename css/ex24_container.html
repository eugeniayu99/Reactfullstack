컨테이너 없이 만든 영역
이 영역은 화면 전체 너비를 그대로 사용한다 큰 모니터에서 보면 줄 길이가 너무 길어서 읽기 불편하다 또한 본문이 왼쪽 위에만 붙어 있어서 디자인적으로 답답해 보인다
컨테이너를 사용한 영역
이 영역은 max-width로 너비가 제한돼서 본문 줄 길이가 적당하다 margin: 0 auto 덕분에 화면 가운데에 배치되어 훨씬 안정적으로 보인다 padding으로 내부 여백을 줬기 때문에 텍스트가 박스 테두리에 붙지 않는다 border-radius와 box-shadow로 카드처럼 보이는 것도 요즘 트렌드 디자인이다
모바일에서 보면 width:100%로 꽉 차고, 데스크탑에서 보면 가운데 정렬된 박스처럼 보인다 이것이 반응형 레이아웃의 기본 구조다
